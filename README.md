A simple command-line habit tracker built with beginner Python syntax. Log your daily habits, mark them as done, and track your progress — all from the terminal.
 
---
 
## 🚀 How to Run
 
```bash
python habit_log.py
```
 
> No libraries needed — uses only built-in Python!
 
---
 
## 📌 Features
 
- ✅ Add new habits
- ✔️ Mark habits as done for today
- 📊 View all habits with status
- 🔄 Reset today's log
- 🚪 Quit anytime
---
 
## 🧠 Python Concepts Used
 
| Concept | Where |
|---|---|
| `list` | `habits = []`, `done_today = []` |
| `def` | `add_habit()`, `mark_done()`, etc. |
| `while` loop | Main menu loop |
| `for` loop | Listing habits |
| `if / elif / else` | Menu choices & validation |
| `input()` | All user prompts |
